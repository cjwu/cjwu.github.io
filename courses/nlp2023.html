Info & Objectives

This introductory course is about a variety of ways to represent human languages for Computer Science (CS) undergraduate students. This course will introduce students to the basics of Natural Language Processing (NLP). The students will gain the ability to exploit languages representations to write programs based on the modern data-driven techniques. This course will include some ideas, including Machine Learning and the recent Deep Learning. There will be an emphasis on rapid prototyping, a useful skill in many other areas.

Topics

Topics covered include:

  • Web crawling and indexes
  • Language modeling
  • Representation learning
  • Word Embeddings
  • Text classification
  • Sequence modeling
  • Machine learning models
  • Deep learning models
